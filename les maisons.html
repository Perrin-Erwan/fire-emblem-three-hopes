Les 3 maisons et autres


Les 3 maisons concernent les 3 pays qui peuplent ce jeu.
Les 3 classes ramènent pratiquement toutes les familles nobles comme les roturiers de ces mêmes pays.
Les aigles de jais, les cerfs d'or, et les lions de saphir.
Les chevalier de Seiros sont des chevaliers de Garreg Mach.
Les mercenaires et esprits sont des personnages importants et indépendants. pour en savoir plus sur les personnages, regardez sur :
  • Les aigles de jais
  • Les lions de saphir
  • Les cerf d'or
  • Loups cendrés
  • Les autres
